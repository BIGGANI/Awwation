Awwation

create Zooming presentations!

Awwation is an app for creating Prezi-like presentations in HTML5, so they can run in modern web browsers without Flash. This is

It is currently in alpha, and the editor works only in Chrome. Presentations can be viewed in the latest versions of Chrome, Firefox, and IE.

Try it!

Make your presentations Awesome.

Features

  • Simultaneous Zooming and Rotations
  • Runs completely in the browser
  • Embed images
  • Supports WebFonts
  • Themes collection
  • Open-Source!
